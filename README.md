West Edmonton Mall: Interactive Sales Digideck
Transforming B2B Presentations into Immersive Digital Journeys
1. Project Vision
The WEM Interactive Sales Digideck is a next-generation B2B presentation tool designed to replace traditional, static slide decks with a dynamic, "video-first" interactive experience. Built for high-value commercial partnerships, the application serves as a non-linear gateway into the West Edmonton Mall ecosystem, emphasizing its scale, luxury, and multi-faceted nature as a "city within a city."

2. Design Philosophy: "Luxury-Minimalist Cinematic"
The aesthetic is crafted to evoke a premium, high-end feel that aligns with luxury retail and international hospitality standards.

Atmospheric Visuals: High-definition video backgrounds, generative ambient light (Aurora glows), and grain textures create a cinematic feel.
Typography: A sophisticated pairing of Cormorant Garamond (serif for elegance) and Inter (sans-serif for modern clarity).
Color Palette: A deep Dark-900 (#0a0a0a) base punctuated by Gold-500 (#d4af37) accents, signifying prestige.
Interactive UI: Custom cursors, glassmorphism (backdrop-blur), and subtle micro-animations make the interface feel alive and responsive.
3. Core Features & User Experience
A. Non-Linear Navigation
Unlike a linear PowerPoint, the Digideck allows presenters to navigate based on client interest:

Central Hub: A navigational "home base" where users can branch off into specific "Pathways."
Pathway System: Dedicated modules for Retail & Luxury, Events, Entertainment, Hospitality, and more.
Interactive Map: A global "Map" modal accessible at any time to provide spatial context of the massive complex.
B. Immersive Transitions
Using Framer Motion, every transition is designed to be fluid and high-end:

Wait-for-Exit: Seamless handoffs between pages using AnimatePresence.
Staggered Reveals: Data points and titles enter the frame with elegant timing to keep the viewer engaged.
C. Data Visualization
Key statistics (e.g., "30M+ Annual Visitors") are not just text but Animated Numbers that count up as they enter the viewport, reinforcing the mall’s immense scale.

4. Technical Architecture
The project is built on a modern, high-performance stack:

Framework: React + Vite for ultra-fast development and optimized build output.
Animation Engine: Framer Motion for complex orchestrated transitions and state-based animations.
Styling: Tailwind CSS 4 with a custom theme configuration for a unified design system.
Icons: Lucide-React for consistent, minimalist iconography.
Navigation: React Router for a smooth, single-page application experience.
5. Project Roadmap & Impact
The Digideck is designed to be a living tool. Its modular "Pathway" structure allows for easy expansion as new attractions or retail zones open at WEM. By moving away from static slides, sales teams can provide a bespoke presentation experience tailored to the specific needs of potential partners, significantly increasing brand perception and conversion potential.
